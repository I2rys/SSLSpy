## Installation
Github:

    git clone https://github.com/I2rys/SSLSpy
    
NPM Packages:
```
npm i express-param && npm i get-domain-certificate && npm i express
```

Test NPM Packages:
```
npm i got
```

## Usage
Default start:
```
node index.js
```

NPM start:
```
npm start
```

## Dev Usage
NPM start dev:
```
npm run start:dev
```

NPM test:
```
npm test
```
